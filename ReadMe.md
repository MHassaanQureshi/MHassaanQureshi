## 👋 About Me

I'm a full-stack developer based in Karachi, building modern web applications with **Next.js, React, TypeScript**, and **FastAPI**. Recently graduated in Computer Science, and currently sharpening my skills through the **Governor Sindh Initiative for AI, Web 3.0, and the Metaverse**.

- 🔭 Currently building full-stack apps with the **MERN stack + Next.js**
- ⚡ Backend work in **FastAPI** for high-performance APIs
- 🤖 Exploring **AI agents, Claude, and generative AI workflows**
- 🎓 CS Graduate — Federal Urdu University of Science & Technology, Karachi
- 🚀 Focused on shipping real, deployed products — not just tutorials
